Ausführliche Beschreibung

Accessor for I/O-Space.

Every computer uses a separate I/O memory that is only adressable by the methods 'in' and 'out'. Each instance of IO_Port is an abstraction of an adress in that memory. Using those abstractions adresses in memory can be accessed like ports.
